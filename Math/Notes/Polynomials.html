Introduction
Before we start learning more about Polynomials, We need to get familiar with a few terms. These are:
➤ Constant
➤ Variable
➤ Term
➤ Algeraic Expression
Let us discuss al of these terms one by one.
Constant :
A Constant is a Fixed Value. It is a value that does not change
For Example : 2, -3, 0 , ⅞ etc.
Variable :
A variable is a symbol that we assign to an unknown value. It is usually represented by letters such as x, y, or t. For example, we might say that l stands for the length of a rectangle and w stands for the width of the rectangle. We use variables when we need to indicate how objects are related even though we may not know the exact values of the objects. For example, if we want to say that the length of a rectangle is 3 times the length of its width then we can write.
Term :
A term is obtained by either multiplication or division of constants and variables.
For Example : 2xy, 2, 5, 5y, 9yzx etc.
Note : Even a single constant or a single variable is a term. for example : 5, x, 10, z are terms.
Algebraic Expression :
Algebraic expression is the combination of terms either by addition or subtraction.
For Example : -10xy + 2y - 2, -8a + 4y², 2 - 10p + 10a².
Now lets start learning about Polynomials.
Polynomials
a Polynomial is an expression consisting of variables and coefficients, that involves only the operations of addition, subtraction, multiplication, and non-negative integer exponentiation of variables.
For Example: x³ + 2xyz² − yz + 1, x² − 4x + 7, x + 50 etc.
Note : A polynomial cannot have a negative exponentiation value.

For Example :

✅ 10x² + 2x + 50 is a Polynomial.

❌ 2x-2 + 9x + 50 is not a polynomial because the term 2x-2 has a negative exponentiation value.
Degree of a Polynomial
The Highest power of the variable in the polynomial is called the degree of the polynomial.

For Example : The degree of the polynomial 6x4 + 2x3+ 3 is 4.
Types of Polynomials (Based on Degree)
Linear Polynomial:
A polynomial whose degree is one ('1') .
For Example : 2x + 3, 4x, x - 10.

The Standard form of a linear polynomial is ax + b where a ≠ 0 and a and b are constants.
Quadratic Polynomial:
A polynomial whose degree is two ('2') .
For Example : x² - 3x + 10.

The standard form of a quadratic polynomial is ax² + bx + c. Where a ≠ 0, and a, b and c are constants.
Cubic Polynomial:
A polynomial whose degree is three ('3') .
For Example : 4x³ - 6x + 60.

The standard form of a cubic polynomial is ax³ + bx² + cx + d. Where a ≠ 0, and a, b, c and d are constants.
